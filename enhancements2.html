Enhancements - JavaScript

  1. Form Error Highlighting
    Using JavaScript I enhanced the functionality of the new validation functions. Creating a function called errorHighlightField, this function accepts an id of an element which needs to be highlighted red.
    This complements the red colour of the error list at the bottom of the form and can easily be cleared using the errorClearField function.
    To test this feature, visit the apply page.
  2. Notifications System
    Using pure JavaScript, I created a simple toast notifications system.
    This is heavily used in the the apply page for showing errors and in the job pages to notify the user that their job preference has been saved. If you want to test this notification system, Click here!
