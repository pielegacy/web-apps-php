PHP Enhancements

  1. Job Description Page Rendering Now Dynamically Handled
    From the start I have not liked having different pages for different jobs as it is messy and not very scalable.
    Thanks to PHP I have changed the structure of the job descriptions to instead all lead to the same page job-description.php.
    Job descriptions are now stored as strings in main.php and are rendered using a function called RenderJobPage. This takes no parameters and instead relies on a variable stored in the url called jobid (eg job-description.php?jobid=DB0001 will take you to DB0001's job page).
